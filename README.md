## ゲームストーリー
超能力(念動力)を持った店主がいるラーメン屋。  
超能力を使いこなしてラーメンを作り、客に提供して満足させる！

## ゲームルール
・操作  
ジャイロで画面を傾けて視点を操作  
画面右側をタップしている間物を掴む  
画面左側上下スライドで物を奥に手前に動かす  
・流れ  
1.麺とスープを入れる  
2.注文に合わせてトッピング  
3.完成したラーメンを提供  
・システム  
制限時間内にスコアを稼ぐ  
正しい注文だと10点加点  
間違った注文だと5点減点
